FAQ and Guides

Frequently Asked Questions How do I add Nano to my server? It's easy. Just click on this link, sign into Discord (if not already) and click Authorize Commands are not working! There are a few possibilities: You're either using the wrong prefix (get it with nano.prefix) or Nano is sleeping (ask an admin to wake him up with nano.wake) A feature is broken! Before reporting a bug check a few things: check if you are using the right prefix (with nano.prefix) and look at !help [your command] to see if you are using the command correctly. If you are, then it would be nice if you could report this bug to me/helpers on the Nano Lounge I have a good idea for a feature! Every suggestion is very welcome and will be considered. When you have a solid idea, make sure to tell me at Nano Lounge or use !suggest [your suggestion] Where can I see upcoming features and known bugs? I'm currently using Trello to track progress on new features, improvements and fixing bugs.
Guides How do I set up Nano's prefix? It's simple.
1. Make sure you have correct permissions: you must be server owner or Nano Admin to use this
2. Run the command nano.changeprefix [your prefix] (where [your prefix] is the new prefix you want). How do I make someone a mod / admin? Nano works on a principle of roles for permissions. To begin, create two roles: Nano Mod and Nano Admin. These roles DO NOT need any permissions, the only important part is the name. To make someone a mod, assign them the role Nano Mod and for admins, give them the Nano Admin role.
A list of permitted commands for each role can be found on the commands page. How do I make Nano ignore a certain channel? Nano has a feature called blacklisting. With it, you can specify channels to ignore.
First off, make sure that you have theNano Adminpermission. Then, simply run this command: nano.blacklist add #your_channel (where your channel is the channel you want to ignore) How do I set up "Join Messages"? For this, you need to be a Nano Admin. Simply run this command: !joinmsg [your message] (replace [jour message] with the actual message).
The following formatting applies: :user -> user's name, :server -> server name
Example: !joinmsg :user, welcome to our amazing server! How do I allow normal members to give themselves a role? (more commonly known as selfroles) Setting up selfroles requires Nano Admin permissions. First off, you must set the role that you want others to be able to access with nano.settings selfrole [role name]. Make sure that Nano has enough permissions (Manage Roles and the role must be lower than Nano's top role).
Then, members simply have to type in the command !selfrole [role name] to get the role (knowing the role name is required).
